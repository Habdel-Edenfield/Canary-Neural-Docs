## 🚀 Como usar?

1. Clone o repositório
    
2. Coloque os arquivos `.hpp` em `/input/src/`
    
3. Execute o script:
    

bash

Copy

python canary_analyzer.py

4. Documentos gerados em `/output/src_new/`
